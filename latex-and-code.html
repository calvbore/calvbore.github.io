Mixed content

The Riemann zeta function is defined as:

\[ \zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s} \]

and for \(s = 2\) we get \(\zeta(2) = \frac{\pi^2}{6}\), which can be computed with:

def zeta_2(terms=10000):
    return sum(1 / (n * n) for n in range(1, terms + 1))
